# kyUI for React

리액트를 위한 UI 컴포넌트 라이브러리입니다.
headless 디자인을 목표로 하고있으면 점진적으로 컴포넌트를 추가해나갈 예정입니다.

### 설치방법

```
// npm
npm install kyui-react

// yarn
yarn add kyui-react
```

### 사용방법

```
import { Button } from 'kyui-react'

const myComponent = () => {
  <div>
    <Button>button</Button>
  </div>
}
```
